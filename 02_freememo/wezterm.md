# WezTerm 操作マニュアル (Mac版)

現在の `wezterm.lua` 設定に基づいたカスタム操作と、WezTermの標準的な操作方法のまとめです。

## 1. 基本コンセプト
* **Leaderキー**: `Ctrl + a`
    * 多くのカスタム操作の起点となります。まず `Ctrl + a` を押し、指を離してから1秒以内に次のキーを入力してください。
* **デフォルトシェル**: `zsh`

---

## 2. ペイン操作 (画面分割・移動)
設定ファイルで定義された Vim 風のキーバインドです。

| 機能 | ショートカット |
| :--- | :--- |
| **左右に画面分割** | `Leader` → `|` (Shift + \) |
| **上下に画面分割** | `Leader` → `-` |
| **左のペインへ移動** | `Leader` → `h` |
| **下のペインへ移動** | `Leader` → `j` |
| **上のペインへ移動** | `Leader` → `k` |
| **右のペインへ移動** | `Leader` → `l` |
| **ペインのリサイズ** | `Leader` → `矢印キー` (5セルずつ) |
| **現在のペインを閉じる** | `Ctrl + d` (または `exit` 入力) |

---

## 3. タブ・ウィンドウ操作
WezTermの標準的な管理機能です。

| 機能 | ショートカット |
| :--- | :--- |
| **新しいタブを作成** | `Cmd + t` |
| **現在のタブを閉じる** | `Cmd + w` |
| **次のタブへ切り替え** | `Ctrl + Tab` |
| **前のタブへ切り替え** | `Ctrl + Shift + Tab` |
| **新しいウィンドウを開く** | `Cmd + n` |
| **フルスクリーン切り替え** | `Cmd + Enter` |

---

## 4. テキスト操作・検索
スクロールバック（履歴）は **3500行** まで保持されます。

| 機能 | ショートカット |
| :--- | :--- |
| **コピー** | `Cmd + c` (または範囲選択) |
| **貼り付け** | `Cmd + v` |
| **画面内検索** | `Cmd + Shift + f` |
| **クイックセレクト** | `Cmd + Shift + space` (画面上のパス等をキー選択) |
| **フォント拡大 / 縮小** | `Cmd + +` / `Cmd + -` |
| **フォントサイズリセット** | `Cmd + 0` |

---

## 5. 管理・ユーティリティ

| 機能             | ショートカット / 補足                     |
| :------------- | :------------------------------- |
| **コマンドパレット**   | `Cmd + Shift + p` (全ての機能を検索可能)   |
| **デバッグオーバーレイ** | `Cmd + Shift + l` (設定のエラー等を確認)   |
| **設定のリロード**    | `Cmd + Shift + r` (手動リロードが必要な場合) |
| **WezTermを終了** | `Cmd + q`                        |

---

## 6. 現在の設定のポイント
* **外観**: テーマ `Paper (Gogh)` を使用。
* **背景**: 不透明度 `0.9` に加え、Mac特有のブラー効果（ぼかし）が `20` に設定されています。
* **自動更新**: 24時間ごとにアップデートを確認します。
* **設定の反映**: `automatically_reload_config = false` のため、設定変更後は WezTerm の再起動または手動リロードが必要です。
